Mutfagram

Kod Mutfağı Temmuz 2012 Node.js + MongoDB + PhoneGAP + HTML5 Workshop

Download this project as a .zip file Download this project as a tar.gz file

Kodmutfağının ilk çalışmasını ürünü olan Mutfagram'a hoş geldiniz.

Günümüzün "kanayan" ucundaki bu teknolojileri hepimiz heyecanla takip ediyoruz. Yaratıcı kodcular, Node.js, MongoDB, PhoneGAP kullanarak HTML5 temelli, çok kullanıcılı deneyimleri inanılmaz derecede hızlı bir şekilde geliştirebiliyorlar.

Bu Kod Mutfağında sosyal sorumluluklarımız çevresinde toplumumuza katkı sağlayabilecek bir sosyal ağ uygulamasını, bu teknolojileri kullanarak geliştiriyor olacağız. Çalışmanın sonuçlarını ve kodları github üzerinden apache açık kaynak lisansı ile de paylaşacağız. Bu kadar kısa zamanda bu kadar çok iş olur mu demeyin, aklınız kesmiyor ise hiç gelmeyin ;-)

Uygulamamız çevremizde iyileşmesini istediğimiz şeylerin mobil uygulamamızı kullanarak GPS ile taglenmiş bir fotoğrafını çekmemizi sağlayacak. Bu resimleri kısa bir mesaj ile sistemimize kaydedeciğiz, Node.js ile yazılmış bir sunucu kullanarak, dokümanlarımızı MongoDB de saklanacak. Oluşan bu mesajları da yazacağımız Web sitemizden, sosyal medyadan, ve de mobil uygulamamızdan takip edebileceğiz.

Node.js Chrome\'un JavaScript motoru ile çalışan çok hızlı, ölçeklenebilir ağ uygulamaları için bir platformdur.

MongoDB ("humongous" dan geliyor) yüksek performanslı açık kaynak kodlu bir NoSQL veri tabanıdır.

PhoneGap Native mobil uygulamalar ve Web teknolojilerinin avantajlarını bir araya getirebilen HTML5 ve JavaScript teknolojilerine dayanan bir mobil platformdur.
